This project is a full-stack Instagram clone built using Node.js, Express.js, EJS, MongoDB, and Tailwind CSS. It replicates core social media functionalities including user authentication, post creation, and profile management.

🚀 Tech Stack

Backend: Node.js, Express.js

Frontend: EJS (Server-Side Rendering), Tailwind CSS

Database: MongoDB (Mongoose ODM)

Authentication: Session-based login system

✨ Features

User registration & login

Secure authentication

Create, view, and delete posts

Profile page with user-specific posts

Responsive UI using Tailwind CSS

MVC project structure

📚 What I Learned

Building RESTful routes with Express

Connecting and managing MongoDB using Mongoose

Implementing authentication & sessions

Structuring scalable backend projects

Server-side rendering using EJS

🎯 Project Purpose

This project was built to strengthen my understanding of full-stack web development and backend architecture by recreating a real-world social media application.
